## Key Concepts to Master

### Modular Arithmetic
- Modulo basics
- GCD & Extended Euclidean Algorithm
- Modular inverse
- Fermat's Little Theorem
- Euler's Totient Function
- Chinese Remainder Theorem
- Primitive Roots
- Discrete Logarithm Problem

### Symmetric Cryptography
- Encryption/Decryption principles
- AES algorithm (SubBytes, ShiftRows, MixColumns, AddRoundKey)
- Block cipher modes (ECB, CBC, CTR, GCM)
- Padding schemes (PKCS#7)
- Authentication (HMAC, GCM)
- Key management

- Basic understanding of: XOR, binary operations, modular arithmetic basics

## Learning Path

### Week 1: Modular Arithmetic
Focus on building mathematical foundations:
- Days 1-2: Basic modular arithmetic concepts
- Days 3-4: Modular inverses and algorithms
- Days 5-7: Advanced topics (CRT, primitive roots, discrete log)

### Week 2: Symmetric Cryptography
Apply math concepts to practical encryption:
- Day 1: Symmetric encryption overview
- Day 2: Deep dive into AES algorithm
- Day 3: Cipher modes and their security implications
- Day 4: Padding and authentication mechanisms
